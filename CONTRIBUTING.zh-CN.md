# 贡献指南

[English](./CONTRIBUTING.md) | [简体中文](./CONTRIBUTING.zh-CN.md)

感谢你愿意为 `opencode-wechat` 做贡献。

## 当前最有价值的贡献

这个项目仍处于早期阶段，所以当前最有帮助的贡献主要包括：

- 安装与使用反馈
- 可稳定复现的缺陷报告
- 文档改进
- 测试覆盖补充
- 运行安全性增强
- 独立打包体验改进

## 提交 PR 前建议

建议尽量做到：

- 先阅读 README 和文档
- 确认改动符合项目当前边界
- 保持改动范围聚焦、便于 review
- 避免无关重构
- 如有行为变化，补充相应说明

## 贡献风格建议

更推荐以下方式：

- 小而清晰的 PR
- 明确的 commit message
- 代码改动同时补充必要文档
- 优先改进清晰度、稳定性和可用性

## 提交 Issue 时建议提供的信息

如果你要提交 issue，建议尽量提供：

- 你预期会发生什么
- 实际发生了什么
- 你执行的命令
- 去掉敏感信息后的配置结构
- 可用的日志输出或截图

## 安全相关问题

如果你的反馈涉及账号安全、登录态处理、审批策略等安全敏感内容，请尽量谨慎描述问题，并避免公开贴出密钥或敏感信息。

## 早期项目说明

由于这个项目仍在向独立开源仓库整理，部分接口、文件路径和工作流在短期内还会继续演进。
