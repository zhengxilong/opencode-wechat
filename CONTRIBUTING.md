# Contributing

[English](./CONTRIBUTING.md) | [简体中文](./CONTRIBUTING.zh-CN.md)

Thank you for considering contributing to `opencode-wechat`.

## What Is Most Helpful Right Now

The project is still in an early stage, so the most valuable contributions are:

- installation feedback
- reproducible bug reports
- documentation improvements
- test coverage
- runtime safety improvements
- standalone packaging improvements

## Before You Open a Pull Request

Please try to:

- read the README and docs first
- confirm the change matches the project scope
- keep changes focused and easy to review
- avoid unrelated refactors
- document behavior changes when needed

## Contribution Style

Please prefer:

- small, well-scoped pull requests
- clear commit messages
- practical documentation updates alongside code changes
- changes that improve clarity, stability, or usability

## Issues

When filing an issue, it helps to include:

- what you expected to happen
- what actually happened
- the command you ran
- the config shape you used, with secrets removed
- log output or screenshots when available

## Security and Safety

If your report touches account safety, login-state handling, approval policy, or other security-sensitive areas, please describe the issue carefully and avoid posting secrets.

## Early Project Note

Because the project is still being structured for standalone open-source release, some interfaces, file paths, and workflows may continue to evolve.
